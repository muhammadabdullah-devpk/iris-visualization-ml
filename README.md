# Iris Dataset Visualization using Python

## 🚀 Overview
This project focuses on exploring and visualizing the famous Iris dataset using Python. It demonstrates basic data analysis and visualization techniques to understand patterns in data.

## 🧠 Objective
The main objective of this project is to perform exploratory data analysis (EDA) on the Iris dataset and visualize relationships between different features of iris flowers.

## 📊 Dataset
The Iris dataset contains measurements of three flower species:
- Setosa
- Versicolor
- Virginica

Features include:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## ⚙️ Workflow
The project follows these steps:
- Load dataset using Pandas
- Inspect and understand data
- Perform statistical analysis
- Create visualizations:
  - Scatter plots
  - Histograms
  - Box plots
- Analyze relationships between features

## 📈 Visualizations Used
- Scatter Plot for feature comparison
- Histograms for distribution analysis
- Box plots for outlier detection

## 🛠 Tech Stack
- Python 🐍
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Results
The visualization helps in understanding how different iris species differ based on petal and sepal measurements. It also shows clear patterns and separability among classes.
